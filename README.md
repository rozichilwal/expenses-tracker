# Expense Tracker Application


A simple expense tracker application that helps users record income and expenses, calculate balance in real time, and analyze spending patterns using a clean and responsive UI.


## Features
- Add income and expense transactions
- View transaction history
- Calculate total income, expenses, and balance
- Data persistence using localStorage
- Responsive design


## Tech Stack
- React
- JavaScript (ES6+)
- HTML5
- CSS3


## How to Run
1. Clone the repository
   git clone <repo-url>

2. Install dependencies
   npm install

3. Start the app
   npm run dev
